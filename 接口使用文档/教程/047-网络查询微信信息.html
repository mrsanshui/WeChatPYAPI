网络查询微信信息

网络查询微信信息

调用方法：

w.network_query_wx_info(
    self_wx="wxid_xxx",
    key="1380000000"
)

参数：

参数 说明
self_wx 当前微信ID
key 要查询的手机号/QQ号/微信号

返回值：

参数 说明
wx_account 微信账号
nick_name 昵称
avatar_url 头像地址
gender 性别
province 省份
area 地区
v3 v3数据（可用来添加好友）

返回示例：

{
    "wx_account": "xxx",
    "nick_name": "xxx",
    "avatar_url": "https://xxxxxxxxxxxxx",
    "gender": "男",
    "province": "xxx",
    "area": "xxx",
    "v3": "v3_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
}
